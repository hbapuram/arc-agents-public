What Arc Agents Does

🔄 Cross-Domain Synthesis

Reads your cycle phase, sleep quality, meals, workouts, and symptoms—then synthesizes them into one coherent recommendation.

🧠 15 Specialized Agents

Each agent focuses on their domain (nutrition, fitness, cycle, care, sleep, etc.) but shares the same health context.

📊 Real-Time Synthesis

Aurora (morning brief) and Council (evening review) give you personalized insights based on your patterns.

🔒 Privacy First

All data stays in your browser. No user database, no servers storing your health info. Completely offline-first.

📱 No Build Step

React SPA that runs in the browser. Flask API that's stateless and serverless. Deploy anywhere, self-host, or run locally.

💡 Learns As You Go

Personalization improves with every log. Arc figures out your patterns and adapts recommendations.

Get Started in 5 Minutes

Option 1: Self-Host

Local: Clone the repo, install Python deps, run python api/index.py, visit http://localhost:5000

Option 2: Deploy to Vercel

Cloud: Fork the repo, push to GitHub, deploy to Vercel, set your API key. Live in 2 minutes.

Option 3: Docker

Container: Build the Docker image, run with your API key. Works anywhere.
